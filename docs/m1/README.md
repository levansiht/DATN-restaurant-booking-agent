# M1 DATN Documentation

Bộ tài liệu này phục vụ mốc `M1` cho đồ án tốt nghiệp theo hiện trạng repo `demo-chat-bot`.

## Tệp tài liệu

- `M1_REPORT.md`: Bản thảo báo cáo đầu kỳ 10-15 trang.
- `M1_REPORT_WORD_READY.md`: Bản báo cáo dài hơn, trau chuốt hơn, có thể đưa vào Word để chỉnh sửa và in nộp.
- `M1_SLIDES.md`: Dàn ý slide bảo vệ 10-12 slide kèm ý nói chính.
- `M1_DIAGRAMS.md`: Sơ đồ khối hệ thống, sơ đồ quy trình đặt bàn và cách giải thích khi bảo vệ.

## Nguyên tắc sử dụng

- Giữ nguyên hiện trạng kỹ thuật đang có trong repo.
- Không trình bày các tính năng chưa hoàn thành như kết quả đã đạt được.
- Các hướng như `payment`, `Telegram`, `admin dashboard` chỉ được nêu ở phần mở rộng.

## Phạm vi hệ thống đang bám theo repo

- Frontend: React + Vite.
- Backend: Django REST Framework.
- Database: PostgreSQL.
- AI: LangChain + OpenAI/Claude.
- Chức năng hiện có: chatbot đặt bàn, danh sách bàn, tìm bàn, tạo booking, tra cứu booking, JWT auth cơ bản.
